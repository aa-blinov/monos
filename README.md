# Система организации заметок (Zed Notes)

Данный репозиторий организован по принципу тематического разделения сфер жизни и работы для удобной навигации и быстрого поиска информации.

## Структура каталогов

- `Работа/` — Все, что связано с профессиональной деятельностью.
    - `EORA/` — Проекты, стратегии и материалы по работе в EORA.
    - `NSTU/` — Коммерческие предложения и взаимодействие с заказчиками.
    - `Исследования_DPI/` — Материалы по анализу пакетов.
- `Проекты_и_Образование/` — Учеба и личные проекты.
    - `Проект_ВШЭ/` — Проект "Стивен" для ВШЭ.
    - `Образование_жены/` — Учебные материалы.
    - `Личные_проекты/` — Собственные технические разработки.
- `Технические_Знания/` — База знаний по технологиям.
    - `ЯМЗ/` — Все о больших языковых моделях и RAG.
    - `Питон/` — Особенности языка и примеры кода.
    - `Копилот/` — Промпты и кейсы использования ИИ-помощников.
- `Документы/` — Релокация и официальные документы.
    - `Цифровой_кочевник/` — Подача на Digital Nomad Residency (CV, мотивационные письма).
    - `ВНЖ/` — Вопросы ВНЖ, РПП и трактовки законов.
- `Личное_и_Творчество/` — Личное развитие и творчество.
    - `Психология/` — Дневники (эмпатия), заметки по психологии.
    - `Идеи/` — Сборник идей для будущих проектов.
    - `Промпты/` — Избранные промпты для работы с ИИ.
- `assets/` — Статические файлы.
    - `images/` — Скриншоты и изображения.
    - `docs/` — Внешние документы (docx, pdf).

## Правила распределения

1. **Работа vs Проекты:** Если деятельность приносит прямой доход или связана с текущим работодателем (EORA, NSTU), она идет в `Работа/`. Если это самообучение или неоплачиваемый проект — в `Проекты_и_Образование/`.
2. **Технологии:** В `Технические_Знания/` кладем только "чистые" знания (гайды, теорию), а не реализацию конкретного рабочего проекта.
3. **Медиафайлы:** Все картинки и тяжелые документы должны находиться в соответствующих папках внутри `assets/`, а в заметках использоваться ссылки на них.
4. **Именование:** Используйте подчеркивания вместо пробелов в названиях папок для совместимости с терминалом, но внутри заметок (в заголовках) можно использовать кириллицу свободно.

## Автоматизация

### Скрипты
- **`format_notes.py`** — Форматирование всех заметок по стандарту mdformat. Запуск: `uv run format_notes.py`.
- **`new_note.py`** — Интерактивное создание новой заметки с древовидной навигацией по категориям, шаблон с фронтматтером. Запуск: `uv run new_note.py`.

Список агентов и их инструкции по управлению базой знаний находятся в файле `Agents.md`.
