# Система организации заметок (Zed Notes)

Данный репозиторий организован по принципу тематического разделения сфер жизни и работы для удобной навигации и быстрого поиска информации.

## Структура каталогов

- `01_Work/` — Все, что связано с профессиональной деятельностью.
    - `EORA/` — Проекты, стратегии и материалы по работе в EORA.
    - `NSTU/` — Коммерческие предложения и взаимодействие с заказчиками.
    - `Research_DPI/` — Исследовательские материалы по глубокому анализу пакетов.
- `02_Projects_and_Education/` — Учеба и личные проекты.
    - `HSE_Stephen/` — Проект "Стивен" для ВШЭ.
    - `Wife_Study/` — Учебные материалы супруги.
    - `Pet_Projects/` — Собственные технические разработки.
- `03_Tech_Knowledge/` — База знаний по технологиям.
    - `LLM/` — Все о больших языковых моделях и RAG.
    - `Python/` — Особенности языка и примеры кода.
    - `Copilot/` — Промпты и кейсы использования ИИ-помощников.
- `04_Documents/` — Релокация и официальные документы.
    - `Digital_Nomad/` — Подача на Digital Nomad Residency (CV, мотивационные письма).
    - `TRP_and_PR/` — Вопросы ВНЖ, РПП и трактовки законов.
- `05_Personal_and_Creative/` — Личное развитие и творчество.
    - `Psychology/` — Дневники (эмпатия), заметки по психологии.
    - `Ideas/` — Сборник идей для будущих проектов.
    - `Prompts/` — Избранные промпты для работы с ИИ.
- `assets/` — Статические файлы.
    - `images/` — Скриншоты и изображения.
    - `docs/` — Внешние документы (docx, pdf).

## Правила распределения

1. **Работа vs Проекты:** Если деятельность приносит прямой доход или связана с текущим работодателем (EORA, NSTU), она идет в `01_Work`. Если это самообучение или неоплачиваемый проект — в `02_Projects_and_Education`.
2. **Технологии:** В `03_Tech_Knowledge` кладем только "чистые" знания (гайды, теорию), а не реализацию конкретного рабочего проекта.
3. **Медиафайлы:** Все картинки и тяжелые документы должны находиться в соответствующих папках внутри `assets/`, а в заметках использоваться ссылки на них.
4. **Именование:** Используйте латиницу в названиях папок для совместимости с терминалом, но внутри заметок (в заголовках) можно использовать кириллицу.

## Автоматизация

### Скрипты
- **`format_notes.py`** — Форматирование всех заметок по стандарту mdformat. Запуск: `uv run format_notes.py`.
- **`new_note.py`** — Интерактивное создание новой заметки с выбором категории, шаблон с фронтматтером. Запуск: `uv run new_note.py`.

Список агентов и их инструкции по управлению базой знаний находятся в файле `Agents.md`.
