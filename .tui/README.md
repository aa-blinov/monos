# Zed Notes TUI

Terminal User Interface для управления системой заметок Zed Notes.

## Запуск

```bash
# Главное TUI приложение (через uv)
make notes-ui

# Или напрямую
uv run .tui/cli.py
```

## Команды

### Основные (в TUI)

| Клавиша | Действие |
|---------|----------|
| `q` | Выход |
| `n` | Новая заметка (Ctrl+N) |
| `/` | Поиск |
| `Ctrl+R` | Недавние заметки |
| `Ctrl+S` | Синхронизировать (Git) |
| `Ctrl+F` | Форматировать заметки |
| `?` | Справка |

### Навигация

| Клавиша | Действие |
|---------|----------|
| `j/k`, `↑/↓` | Вверх/вниз |
| `h/l`, `←/→` | По папкам |
| `Enter` | Открыть файл |
| `e` | Редактировать в $EDITOR |

### CLI опции

```bash
# Запустить TUI
uv run .tui/cli.py

# Создать новую заметку (интерактивно)
uv run .tui/cli.py --new

# Поиск
uv run .tui/cli.py --search "react"

# Показать недавние
uv run .tui/cli.py --recent

# Форматировать
uv run .tui/cli.py --format

# Синхронизировать с Git
uv run .tui/cli.py --sync

# Статистика
uv run .tui/cli.py --stats
```

## Описание модулей

### `app.py`

Главное Textual приложение с экранами:
- **MainScreen** — главный экран с браузером заметок
- **NewNoteScreen** — создание новой заметки
- **SearchScreen** — поиск заметок
- **HelpScreen** — справка

### `models.py`

**NotesManager** — управление заметками:
- Загрузка заметок
- Поиск
- Создание новых заметок
- Git синхронизация
- Форматирование
- Статистика

### `widgets.py`

Пользовательские виджеты:
- **NotesBrowser** — браузер заметок в виде дерева
- **SearchWidget** — поиск (WIP)
- **StatusBar** — статус-бар

### `cli.py`

CLI интерфейс для работы из командной строки:
- Интерактивные режимы
- Batch операции
- Help и версия

## Развитие

### Режим разработки

```bash
make notes-ui-dev
```

Запускает TUI с поддержкой горячей перезагрузки.

## Замена старых скриптов

TUI заменяет следующие скрипты через опции CLI:

- `.scripts/new_note.py` → `uv run .tui/cli.py --new`
- `.scripts/format_notes.py` → `uv run .tui/cli.py --format`
- `.scripts/check_links.py` → (интегрировано в TUI)

Старые скрипты остаются для обратной совместимости.
