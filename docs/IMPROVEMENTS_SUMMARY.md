## 🔧 Созданные инструменты для автоматизации

### 1. `.scripts/improve_articles.py`
- Проверяет наличие сломанного текста
- Проверяет наличие citations
- Проверяет наличие Table of Contents
- Выводит статистику структуры (разделы, подразделы)

**Использование:**
```bash
python3 .scripts/improve_articles.py
```

### 2. `.scripts/enhance_articles.py`
- Добавляет Claude-specific guidance в статьи
- Добавляет дополнительные citations где их недостаточно
- Улучшает практические рекомендации

**Использование:**
```bash
python3 .scripts/enhance_articles.py
```

**Результат:** Добавлен совет для Claude в раздел "Лучшие практики" статьи про агентов

### 3. `.scripts/check_links.py`
- Проверяет все типы ссылок: markdown, HTTP(S), arXiv citations
- Проверяет внутренние якоря
- Проверяет наличие файлов
- Выводит подробный отчёт

**Использование:**
```bash
python3 .scripts/check_links.py
```

**Результат:**
```
✅ Все ссылки в порядке!
- Проверено файлов: 2
- Обнаружено проблем: 0
```
