Local-first notes for focused thinking

Your knowledge base, on your machine.

Monos is a desktop notes app built around Markdown, wiki links, fast search, and optional Git sync. It keeps your notes portable while giving you a polished workspace for daily thinking.

Download builds View source

Built for private, durable notes.

Monos treats files as the product. The interface is there to make those files easier to create, connect, search, and sync.

Plain Markdown

Your notes stay readable outside the app. SQLite is only a rebuildable index.

Wiki links

Connect notes with `[[links]]` and use backlinks to navigate your knowledge graph.

Desktop storage

User data lives in the operating system's app data directory, not in the source repo.

Board view

A compact dashboard helps you scan recent notes and create new ones quickly.

Full-text search

Search titles, tags, and body content with highlighted matches.

Optional Git sync

Use your own repository for backup and history when you want it.

Small details that make notes feel safe.

Monos keeps the technical model simple, but the product benefits are visible every day: files stay yours, search stays fast, and sync stays optional.

01 Local-first storage

Notes are plain Markdown files in the OS app data folder, not hidden inside a cloud account.

02 Rebuildable index

SQLite speeds up search and recents, but Markdown remains the source of truth.

03 Fast daily capture

The board keeps recent work visible and makes quick notes feel like part of the same workspace.

04 Optional sync

Git backup is available when you want history, but the app works without remote services.

Electron

Owns the desktop window and user data location.

Local API

Reads and writes notes through localhost only.

Markdown

Keeps the durable, portable note format.

Git

Adds user-controlled backup and history.

Monos is open source and local-first. Your notes belong to you.
